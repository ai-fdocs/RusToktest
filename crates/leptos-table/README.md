# leptos-table

## Назначение
`leptos-table` — минималистичные хелперы для создания таблиц в Leptos-приложениях.

## Что делает
- Упрощает создание таблиц данных в Leptos.
- Предоставляет компоненты для заголовков, строк и ячеек.
- Поддерживает сортировку и базовую фильтрацию.
- Позволяет настраивать отображение колонок.

## Как работает (простыми словами)
1. Определяется структура таблицы (колонки и их свойства).
2. Данные передаются в таблицу как вектор строк.
3. Таблица автоматически рендерит заголовки и данные.
4. При клике на заголовок можно включить сортировку.
5. Поддерживается кастомный рендеринг ячеек через слоты.

## Ключевые компоненты
- `table/` — основной компонент `Table`.
- `column/` — определение колонки `Column` с настройками.
- `row/` — компонент строки таблицы.
- `cell/` — компонент ячейки.
- `sort/` — логика сортировки колонок.

## Кому нужен
Для фронтенд-приложений на Leptos, отображающих табличные данные:
- `apps/admin` — таблицы контента, заказов, пользователей.
- `apps/storefront` — списки товаров, история заказов.

This is an alpha version and requires clarification. Be careful, there may be errors in the text. So that no one thinks that this is an immutable rule.
