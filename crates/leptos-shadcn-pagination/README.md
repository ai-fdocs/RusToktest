# leptos-shadcn-pagination

## Назначение
`leptos-shadcn-pagination` — локальный патч компонента пагинации из shadcn-ui для Leptos.

## Что делает
- Реализует компонент пагинации в стиле shadcn/ui.
- Поддерживает навигацию по страницам (вперёд/назад, номера страниц).
- Предоставляет стилизованные кнопки для управления пагинацией.
- Интегрируется с дизайн-системой проекта.

## Как работает (простыми словами)
1. Компонент принимает текущую страницу и общее количество страниц.
2. Рендерит навигацию с кнопками "Назад", номерами страниц, "Вперёд".
3. При клике на кнопку вызывается callback с новым номером страницы.
4. Стили применяются через CSS-классы в стиле shadcn/ui.

## Ключевые компоненты
- `pagination/` — основной компонент `Pagination` и вспомогательные компоненты.
- `page_button/` — кнопка отдельной страницы.
- `navigation/` — кнопки навигации (вперёд/назад).

## Кому нужен
Для фронтенд-приложений на Leptos, использующих дизайн-систему shadcn/ui:
- `apps/admin` — пагинация в таблицах и списках.
- `apps/storefront` — пагинация каталога товаров.

This is an alpha version and requires clarification. Be careful, there may be errors in the text. So that no one thinks that this is an immutable rule.
