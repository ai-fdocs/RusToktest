# leptos-hook-form

## Назначение
`leptos-hook-form` — минималистичные хелперы для управления состоянием форм в Leptos на основе хуков.

## Что делает
- Предоставляет лёгковесный подход к управлению формами через хуки.
- Не требует контекста — работает на уровне отдельных полей.
- Поддерживает базовую валидацию полей.
- Интегрируется с нативными HTML-формами.

## Как работает (простыми словами)
1. Хук `use_form_field` создаёт реактивное состояние для одного поля.
2. Каждое поле отслеживает своё значение, ошибки и состояние.
3. Валидация выполняется при изменении значения или отправке формы.
4. Хук `use_form` объединяет несколько полей в единую форму.
5. Данные собираются в структуру при отправке.

## Ключевые компоненты
- `field/` — хук `use_form_field` для управления отдельным полем.
- `form/` — хук `use_form` для объединения полей.
- `validation/` — простые валидаторы для полей.

## Кому нужен
Для простых форм в Leptos-приложениях, где не нужен полноценный `leptos-forms`:
- Минимальные формы поиска.
- Быстрые прототипы.
- Случаи, когда важен размер бандла.
